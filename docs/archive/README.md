# Archive Folder

This folder contains documentation and diagrams from the original research project that this package was extracted from.

These files are kept for historical reference but are not relevant to the current `mlflow-eval-tools` package.

## Contents

- `architecture.md` - Original system architecture documentation for the full LLM lifecycle platform research project
- `*.svg` - System diagrams (C1, C2, C3 context diagrams)
